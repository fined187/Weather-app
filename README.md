# About The Project!
---
##### This Project is for making simple Weather-app using React-Native(RN). Nowadays so many projects(like iPhone or Android app) were made by Flutter. Before making app with Flutter, i just tried to know what is the difference between RN and Flutter. So first of all i made this app with using RN, and then with using Flutter. And i will analyze the differences and learn those two programming languages.
##### This project is showing users location, temperature and weather. Users can swipe the screen and can get the nearby town's wheather.
